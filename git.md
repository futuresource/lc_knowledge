#### 生成key
```sh-keygen -t rsa -C "youremail@example.com"```

#### 生成git仓库
```git init```

#### 添加文件
```git add something```

#### 备注提交文件注释
```git commit -m "something"```

#### 添加远程仓库地址
```git remote add origin git@github.com:username/projectname.git```

#### 提交到远程仓库
```git push -u origin master```
